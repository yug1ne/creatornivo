POST STRUCTURE

Use a structure appropriate to the post type.

GENERAL UPDATE

1. State what changed.
2. Explain who it affects.
3. Include timing or location.
4. Provide one action.

OFFER

1. State the exact offer.
2. State eligible product, service, or customer.
3. State end date and essential conditions.
4. Provide one redemption action.

EVENT

1. State event name.
2. Include date, time, time zone, and location.
3. Explain who it is for.
4. State registration requirement.
5. Provide one registration action.

SPECIAL HOURS

1. State the exact date.
2. State changed hours.
3. Explain affected services.
4. State when normal hours resume.

CLOSURE OR INTERRUPTION

1. State affected location or service.
2. State start and expected end when confirmed.
3. Explain available alternatives.
4. Provide the next update or support destination.

PRODUCT OR SERVICE ANNOUNCEMENT

1. State what is now available.
2. Explain who it helps.
3. Mention the most relevant confirmed feature or condition.
4. State availability.
5. Provide one action.
